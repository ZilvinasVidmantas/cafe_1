Pagrindinės CSS savybės

Float

CSS float savybį gali slinkti į vieną iš dviejų pusių: kairę, arba dešinę.
Po slenkančiais(float) elementais, kiti display block elementai, jų tarsi nepripažįsta-nemato, todėl slenekantieji (float) elementai įsiveržia į po apačia esančių display: block elementų kontūrus.
Visgi, po slenkančiais(float) elementais aprašytuose display: block elementuose esantis tekstas, nepersidengia su slenkančiais elementais, o aplink juos "apteka".

Clear

css savybė clear atpažįsta viršuje esančius float elementus ir neleidžia jiems patekti į display: block elementų rezervuojamą vietą

Galimos 'clear' savybės reikšmės
  • left - atpažįsta elementus kurie slenka į kairę
  • right - atpažįsta elementus kurie slenka į dešinę
  • both - atpažįsta visus slenkančius elementus

Position

Position tipai
  • static (default) - Elementai pozicionuojami iš kairės į dešinę, jei netelpa į tėvinį elementą - persikelia į sekančią eilutę
  • relative - elementas pozicionuojamas pagal jo menamąją static poziciją. Naudojant savybės top, right, bottom, left gali paslinkti elementą, nuo jo menamosios vietos
  • absolute - elementas ieško pirmo tėvinės hierarchijos elemento, kurio position savybė NĖRA static. Radęs jį, atsispiria nuo jo kontūrų pagal savybes top, right, bottom, left
    Jeigu tėvinės hierarchijos šakoje nėra elementų, kurie būtų NE static, tuomet position absolute elementas atsispiria nuo viewport'o.
    Kiti elementai nemato elemento su position absolute ir lygiuojasi tarsi jo nebūtų.
  • sticky - išlaiko poziciją pagal pirmo tėvinėje hierachijoje randamo scroll'inamo elementom naudojant savybes top, right, bottom, left, kuomet tėvinis elementas yra scroll'inamas.
  • fixed - elementas atsispiria nuo vieport'o pagal savybes top, right, bottom, left.
    Kiti elementai nemato elemento su position fixed ir lygiuojasi tarsi jo nebūtų.
