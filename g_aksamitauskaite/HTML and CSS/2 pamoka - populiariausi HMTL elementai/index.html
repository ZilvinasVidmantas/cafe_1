Pagrindiniai HTML elementai

Jūs patys galite kurti HTML elementus, bet tam, kad būtų geriau dirbti komandoje yra naudojami standartiniai. Kai kuriems elementams naršyklė suteikia stilius pagal tag'o pavadinimą, tuo tarpu ir išdėstymo tipą.

  • block - tai elementai kurie užsirezervuoja visą tėvinio elemento plotį
  • inline-block - tai elementai kurie gali turėti 2D dimensijų nustatymus ir yra apibrėžti
  • inline - tai tekstas

Nuorodos

Nuorodos kuriamos naudojant <a> tag'ą. Anliškai - anchor tag.
Norint nurodyti nuorodos vietą, turime sukurti atributą href="..."
Jeigu norite kad puslapis atsidarytu naujame skirtuke, turite papildomai parašyti atributą target="blank"
Taip pat nuorodos, gali nukreipti ne tik į kitą puslapį, bet ir į šio puslapiop sekciją. Norint tai padaryti turime sukurti sekciją su atributu id="..." ir tuomet nuorodos href atribute parašyti simbolį '#' ir pridėti elemento į kurį norime nukreipti id

Google - be target atributo. Atidaro šiame skirtuke.

Nuotraukos

Nuotraukoms yra naudojamas <img> elementas. Tam kad nurody kelią iki nuorodoas naudojamas src="..."atributas
Taip pat labai svarbu yra nurodyti alt atributą. Šis atributas parašo tekstą, jeigu nuotrauka nėra randama. Jį reikėtų nurodyti, netgi jeigu esate įsitikinę, jog nuotrauka bus tikrai rasta. Kai kurie žmonės būna išjungė nuotraukų rodymą savo naršyklėje, ir tuomet tekstas padeda jiems suprasti kas toje vietoje turėjo būti atvaizduojama. Dar viena svarbi priežasti yrs - SEO. Pagal alt atribute aprašytą tekstą, yra iškeliamas jūsų puslapio paieškomumas. Todėl reikėtų alt atribute kuo tiksliau įvardinti nuotraukos turinį, naudojant iškeliamuosius raktažodžius.

Įvesties elementai

Įvesties elementai naudojami tam, kad vartotojas galėtų įvesti duomenis.
Beveik visada įvesties elementai yra apgaubiami <form> elementu.
Forma, tai įvesties laukų grupė, kuri turi galimybę surinkti visus duomenis esančius viduje ir išsiųsti juos reikiamu adresu.
Norint, jog būtų siunčiami įvesties lauko duomenys, submit'inant formą, turi suteikti įvesties laukui name atributą.
Norint įvesties lauką padaryti privaloma turime uždėti required atributą.

  • method - šiame atribute nurodoma, kokiu būdu siunčiami duomenys į serverį.

Pomėgiai

Block elements

Blokiniai elementai, tai tokie elementai kurie atvaizduojami vienas po kitu, lyg namo aukštai. Nepriklausomai nuo jų užimamo turinio, jie užsirezervuoja visą tėvinio elemento plotį.

Dažniausiai naudojami blokiniai elementai:

  • div - standartinis blokinis elementas
  • section - blokinis elementas skirtas puslapio sekcijai
  • footer - blokinis elementas skirtas apatinei sekcijai
  • header - blokinis elementas skirtas antraštei kurti
  • article - blokinis elementas skirtas puslapio straipsniui (sekcijos daliai)

Inline-block elements

Tai atvaizdavimo būdas, kuomet elementai gali turėti margin ir padding, bei aukštį ir plotį. Šie elementai atvaizduojami teksto rašymo kryptimi, iš kairės į dešinę ir iš viršaus į apačią. Jeigu elementas netelpa į tėvinį konteinerį, tuomet jis persikelia į sekančią eilutę.
P.S.: Yra kaip display: block, tik neužsirezervuoja viso tėvinio elemento pločio.

Dažniausiai naudojami inline: block elementai

  • input
  • select
  • textarea

Inline elements

Display inline elementai yra traktuojami kaip tekstas. Jie visuomet užima tik tiek turinio, kiek juose užimatis turinys. Nustačius plotį ar aukštį, jų dydis nepasikeičia. Taip pat neveikia margin ir padding nustatymai vertikalia kryptimi.
inline yra standartinė display CSS savybės reikšmė

Dažniausiai naudojami inline elementai:

  • span - standartinis inline elementas
  • a - nuoroda
  • img - nuotrauka
